What is GitHub?

GitHub is a development platform inspired by the way you work. From open source to business, you can host and review code, manage projects, and build software alongside 40 million developers.

What we will use GitHub for?

GitHub will allow us to do several things:

  • 1.

    GitHub will version control all of your code for Web 2. Which means you might make a mistake and not notice that mistake for hours or even days. If you project is in GitHub you can hop back into the commits and locate at what point you made this mistake.

  • 2.

    GitHub will allow you to share your repo with me and I will be able to branch your work and answer your questions about your code.

  • 3.

    GitHub will allow you to be anywhere on planet Earth ( even though everyone is stuck at home ), login to your GitHub account and have all the files you need to keep working.

Step 5

Create up a new repo

Give the repo a name ( you cannot have any spaces in the name), you don't have to add a description, always set the repo to public ( so that you can share the repo with me if needed ) and check Initialize this repository with a README ( this will add a file into the repo on the next step and make cloning the repo easier for you ) .
